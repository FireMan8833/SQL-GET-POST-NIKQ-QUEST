# Кодовый Шторм: Протокол Формы

Статический браузерный квест по материалам PHP + SQL: формы, GET/POST, `$_GET`, `$_POST`,
SQL injection, экранирование, хеширование паролей, `password_hash`, `password_verify`,
cookies и связи таблиц.

## Что внутри

- 40 теоретических заданий.
- 40 сюжетных переходов с эффектом печати текста.
- Главная героиня: агент Ника.
- Счетчик очков и индикатор щита.
- Подсказки и штраф за ошибку.
- Часть заданий требует ручной ввод текста, а не выбор из вариантов.
- Аркадная мини-игра после 20-го задания.
- Финальная аркадная мини-игра после 40-го задания.
- Локальная музыка через Web Audio API, без внешних аудиофайлов.
- Вся графика хранится в проекте: логотип, SVG-герой, финальная заставка, canvas-эффекты.

## Как запустить локально

Самый простой способ:

1. Откройте файл `index.html` в браузере.
2. Нажмите `Запустить миссию`.

Дополнительный способ через локальный сервер:

```powershell
python -m http.server 8000
```

После этого открыть `http://localhost:8000`.

## 🎓 О проекте

Проект разработан специально для международной ИТ-школы **KIBERone** (КИБЕРone) в качестве интерактивного тренажера по веб-безопасности, PHP и базам данных MySQL.

## 🔑 Секретный код доступа (Чит-код 777)

В игре предусмотрен секретный режим администратора. Чтобы открыть окно ввода чит-кода:
1. Нажмите клавишу **`P`** (или русскую **`П`**) на клавиатуре (убедившись, что курсор не находится в поле ввода) либо нажмите комбинацию **`Ctrl + Alt + K`** (или кликните 5 раз по логотипу KIBERone в шапке).
2. В появившемся системном окне введите код **`777`** и нажмите Enter (или кнопку «Применить»).
3. Это позволит мгновенно проходить текущий сектор или аркадный шлюз!
