# Кодовый Шторм: Протокол Формы

Статический браузерный квест по материалам PHP + SQL: формы, GET/POST, `$_GET`, `$_POST`,
SQL injection, экранирование, хеширование паролей, `password_hash`, `password_verify`,
cookies и связи таблиц.

## Что внутри

- 40 теоретических заданий.
- 40 сюжетных переходов с эффектом печати текста.
- Главная героиня: агент Ника.
- Счетчик очков и индикатор щита.
- Подсказки и штраф за ошибку.
- Часть заданий требует ручной ввод текста, а не выбор из вариантов.
- Аркадная мини-игра после 20-го задания.
- Финальная аркадная мини-игра после 40-го задания.
- Локальная музыка через Web Audio API, без внешних аудиофайлов.
- Вся графика хранится в проекте: логотип, SVG-герой, финальная заставка, canvas-эффекты.

## Как запустить локально

Самый простой способ:

1. Откройте файл `index.html` в браузере.
2. Нажмите `Запустить миссию`.

Дополнительный способ через локальный сервер:

```powershell
python -m http.server 8000
```

После этого открыть `http://localhost:8000`.

## 🎓 О проекте

Проект разработан специально для международной ИТ-школы **KIBERone** (КИБЕРone) в качестве интерактивного тренажера по веб-безопасности, PHP и базам данных MySQL.

## 🔑 Секретные режимы навигации и чит-коды

В проекте предусмотрено два системных окна для быстрой навигации и демонстрации материала:

### 1. Панель быстрой навигации (Главы 1–8 и Чит-код 777)
Позволяет мгновенно перемещаться к любой из 8 глав учебной программы или автоматически завершать текущее задание.
* **Как открыть**:
  - Нажмите комбинацию **`Ctrl + W`** или **`Alt + W`** на клавиатуре.
  - Нажмите клавишу **`W`** (или русскую **`Ц`**), убедившись, что курсор не находится в поле ввода.
  - Кликните по кнопке **`[Навигация: W]`** в подвале (футере) сайта.
* **Доступные команды**:
  - **`1` – `8`**: Быстрый переход к соответствующей главе. Все предыдущие задания и аркады автоматически отмечаются как пройденные, и начисляются соответствующие баллы.
  - **`777`**: Супер-код для мгновенного успешного прохождения текущего сектора или аркадного шлюза с получением максимальных очков.

### 2. Системное окно пошагового доступа
Классическое окно для пошагового продвижения вперед.
* **Как открыть**:
  - Нажмите клавишу **`P`** (или русскую **`П`**), либо комбинацию **`Ctrl + Alt + K`**.
  - Кликните 5 раз подряд по логотипу KIBERone в шапке сайта.
* **Код доступа**: Введите системную сигнатуру (например, `330`) для перехода к следующему этапу.
